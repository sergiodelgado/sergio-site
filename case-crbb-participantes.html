Data Visualization

Mapa interactivo de participantes · Laboratorio de Articulación Creativa

Este caso visualiza datos anonimizados y seguros del formulario de inscripción para explorar territorio, disciplinas, motivaciones y perfiles del ecosistema cultural. La vista prioriza lectura estratégica y uso público, excluyendo identificadores sensibles.

Uso de datos

Este tablero utiliza información proveniente de un formulario de inscripción para un programa cultural. Los datos han sido procesados para fines analíticos y se presentan en formato agregado o público.

  • No se incluyen datos sensibles (RUT, correo, teléfono).
  • Los textos han sido resumidos para proteger privacidad.
  • Las redes sociales mostradas corresponden a perfiles públicos.

Nota metodológica

Datos observados desde formulario

  • Nombre público o artístico, organización, ciudad, región, territorio y redes públicas.
  • Áreas disciplinares, motivaciones declaradas y perfil interdisciplinario.

Visualización derivada

  • Conteos, cruces y agregaciones para lectura territorial y de perfiles.
  • Mapas comparativos de motivaciones por región y macroperfil.

Exclusión de datos sensibles

  • No se publica rut, email, teléfono ni texto completo de postulación.
  • Solo se expone resumen seguro de postulación cuando existe en formato público.
