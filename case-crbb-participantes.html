Data Visualization

Mapa interactivo de participantes · Laboratorio de Articulación Creativa

El tablero permite explorar composición territorial, perfiles, disciplinas y motivaciones declaradas por participantes del Laboratorio de Articulación Creativa. Los datos se usan para lectura analítica del ecosistema, no para contacto comercial ni exposición de información sensible.

Resumen ejecutivo

  • El Biobío concentra el núcleo principal de participación, pero existe alcance interregional.
  • El grupo combina creación artística, gestión cultural y capacidades técnicas.
  • La formación, las redes y la profesionalización aparecen como motivaciones dominantes.

Nota metodológica

Fuente y alcance

  • La página usa un dataset público seguro derivado de un formulario de inscripción cultural.
  • El análisis se limita a conteos, cruces y agregaciones para lectura territorial, disciplinar y motivacional.

Campos visibles

  • Nombre público o artístico, organización declarada, territorio, áreas, motivaciones, tipo de perfil y redes públicas.
  • Los textos de postulación se muestran solo como resumen seguro cuando existe.

Privacidad

  • No se publica RUT, email, teléfono ni texto completo de postulación.
  • La visualización evita usar la información para contacto comercial o exposición de datos sensibles.
