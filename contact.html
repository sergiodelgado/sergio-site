Resolver un problema técnico

Cuéntame el proceso, fuente de datos o decisión que necesitas ordenar. Respondo mejor cuando el problema tiene contexto operativo y un entregable claro.

Problemas típicos

  • Datos dispersos entre planillas, formularios, registros y conversaciones.
  • Procesos no trazables o dependientes de coordinación manual.
  • Reportabilidad manual que consume tiempo y no escala.
  • Baja gobernanza de datos, roles, reglas o criterios de privacidad.
  • Necesidad de dashboards, automatización o IA aplicada.

Entregables posibles

Diagnóstico técnico Arquitectura de datos Dashboard ejecutivo Automatización de flujo Validación QA Documentación operativa
